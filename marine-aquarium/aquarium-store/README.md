## 🏪 店面运营要点

### 活体管理
| 类别 | 管理要点 |
|-----|---------|
| 海水鱼 | 水质、温度、喂食、疾病观察 |
| 珊瑚 | 光照、水流、钙/镁/钾补充 |
| 无脊椎 | 饵料、躲避空间、水质敏感度 |

### 设备维护
| 设备类型 | 维护频率 | 检查要点 |
|---------|---------|---------|
| 蛋白分离器 | 每周 | 清洗收集杯 |
| 钙反应器 | 每月 | 检查材料、调节出水 |
| 灯具 | 每月 | 清洁灯罩、检查光谱 |
| 水泵 | 每月 | 清洁转子、检查噪音 |
| 温控设备 | 每季 | 清洁滤网、校准温度 |
