## 一、寄生虫病

### 1. 海水白点病 (Marine Ich / Cryptocaryon irritans)

#### 识别特征
- **外观：** 鱼体、鳍、鳃上出现小白点，像盐粒
- **行为：** 呼吸急促、蹭缸、呆滞、食欲下降、鳍紧缩
- **晚期：** 眼睛浑浊

#### 生命周期
- 卵囊期 → 释放幼虫 → 附着鱼体 → 成虫脱落 → 繁殖
- **关键：** 成虫脱落后在底床繁殖，必须阻断循环

#### 治疗方案

**A. 铜剂治疗（最有效）**
| 铜剂类型 | 治疗浓度 | 备注 |
|---------|---------|------|
| 离子铜（Cupramine） | 0.5 ppm | 需逐渐升至治疗浓度 |
| 螯合铜（Coppersafe） | 2.0 ppm | 可直接达到起始浓度 |
| Copper Power | 2.2-2.3 ppm | 48小时内达到 |

- **治疗期：** 30天（从达到治疗浓度开始计算）
- **监测：** 每日至少检测2次铜浓度
- **设备：** 使用Hanna HI702铜测试仪

**B. 低盐度治疗（Hyposalinity）**
- **目标盐度：** 1.009-1.010（14-16 ppt）
- **降盐速度：** 每日降0.002，48小时内达到目标
- **治疗期：** 4-6周
- **温度：** 75-80.6°F（24-27°C）
- **注意：** 不可与铜剂同时使用

**C.  tank转移法（TTM）**
- 每3天将鱼转移到新缸，共转移5次（15天）
- 原缸空置11周（77天）饿死寄生虫

**D. 检疫缸设置**
- 裸缸（无底砂）
- 简单过滤（海绵滤或HOB）
- 加热器维持76-78°F
- 气泵增氧
- PVC管提供躲藏处

---

### 2. 海水丝绒病 (Marine Velvet / Amyloodinium ocellatum)

#### 识别特征
- **外观：** 鱼体覆盖金/锈色细粉状涂层，比白点小且密集
- **行为：** 极度呼吸急促、蹭缸、躲藏、畏光、聚集在造流处
- **危险：** 比白点病进展更快，可迅速致死

#### 治疗方案
- **铜剂：** 同白点病治疗浓度
- **氯喹磷酸盐：** 非常有效
- **淡水浴：** 临时缓解，可冲掉寄生虫
- **福尔马林浴：** 有效但需小心使用
- **空置期：** 主缸至少空置6周（42天）

---

### 3. 吸虫/鳃虫 (Flukes / Monogenean Trematodes)

#### 识别特征
- 鱼频繁蹭缸、抽搐
- 呼吸急促、呆滞、食欲下降
- 鳃部红肿、发炎
- 体表可见半透明扁平小虫

#### 确认方法
- **淡水浴测试：** 5-10分钟淡水浴，虫体变白脱落

#### 治疗方案
- **吡喹酮（Praziquantel）：** 最有效
  - 产品：PraziPro
  - 可添加水中或浸泡饲料
- **福尔马林浴**
- **淡水浴**

---

### 4. 其他寄生虫

#### 黑点病（Black Ich / Turbellarians）
- **外观：** 鱼体出现黑色小点
- **治疗：** 吡喹酮、淡水浴、福尔马林

#### 体内寄生虫
- **症状：** 腹部凹陷、白色线状粪便、消瘦但进食
- **治疗：**
  - Seachem MetroPlex（甲硝唑）
  - API General Cure
  - Fritz Expel-P（左旋咪唑）

## 二、细菌感染

### 常见致病菌
- *Vibrio*（弧菌）
- *Aeromonas*（气单胞菌）
- *Pseudomonas*（假单胞菌）
- *Streptococcus*（链球菌）

### 症状识别
- 皮肤红斑、溃疡、开放性伤口
- 鳍条腐烂（烂鳍病）
- 眼睛浑浊或突出（突眼病）
- 腹部肿胀（腹水病），鳞片竖立如松果
- 呆滞、食欲下降
- 呼吸急促

### 治疗方案

| 药物 | 用途 | 使用方法 |
|-----|------|---------|
| **Kanamycin（卡那霉素）** | 广谱抗生素 | Seachem KanaPlex |
| **Erythromycin（红霉素）** | 革兰氏阳性/阴性菌 | API E.M. Erythromycin |
| **Maracyn/Maracyn-2** | 烂鳍、突眼、腹水 | 米诺环素 |
| **Metronidazole（甲硝唑）** | 厌氧菌、内寄生虫 | MetroPlex |
| **磺胺甲噁唑/甲氧苄啶** | 广谱复方 | Fix Sulfa Tablets |

- **药饵：** 内感染最有效，药物通过消化道吸收
- **水体用药：** 外感染直接加药到水中
- **盐浴：** 对某些细菌（如烂鳍）有效

---

## 三、真菌感染

### 识别特征
- 棉絮状白色/灰色/黄色生长物
- 通常发生在伤口处
- 呆滞、食欲下降
- 常伴随细菌感染

### 治疗方案
- **API Fungus Cure**
- **孔雀石绿/亚甲蓝**
- **KanaPlex**（对真菌和细菌都有效）
- **API Pimafix/Melafix组合**
- **淡水浴** - 轻度外真菌有效

---

## 四、综合防治策略

### 1. 预防（最重要）

#### 检疫程序（QT）
- **新鱼隔离期：** 至少4-6周（推荐12周）
- **检疫缸设置：**
  - 20加仑（<4英寸鱼）或40加仑（大鱼）
  - 裸缸、简单过滤、加热、增氧
  - 提前在主力缸"养"过滤棉建立菌群

#### 日常维护
- 保持优质水质
- 定期换水（10-20%，每周）
- 避免过度拥挤
- 减少压力源
- 多样化营养饲料
- UV杀菌灯（减少水中游离寄生虫）

### 2. 治疗原则

1. **早发现早治疗**
2. **确诊再用药** - 避免误诊误治
3. **隔离治疗** - 保护主缸和珊瑚
4. **完成疗程** - 不可症状缓解就停药
5. **监测水质** - 治疗期间更频繁检测

### 3. 药物使用注意事项

- **铜剂：** 对所有无脊椎动物剧毒，绝不可用于含珊瑚的主缸
- **抗生素：** 可能破坏生物过滤，需监测氨/亚硝酸盐
- **混合用药：** 需谨慎，避免药物相互作用
- **温度：** 治疗期间保持77-80°F

## 五、常见鱼种易感性

| 鱼种 | 易患病 | 备注 |
|-----|--------|------|
| 粉蓝吊/粉棕吊 | 白点病 | "白点磁铁" |
| 七彩吊 | 白点病 | 高敏感性 |
| 小丑鱼 |  Brooklynella | 需特别关注 |
| 海水神仙鱼 | 多种寄生虫 | 较难饲养 |

## 六、紧急处理流程

```
发现病鱼
   ↓
立即隔离到检疫缸
   ↓
观察症状 → 拍照记录
   ↓
确诊病因（必要时显微镜检查）
   ↓
选择治疗方案
   ↓
开始治疗 + 每日监测
   ↓
完成疗程 → 观察期
   ↓
确认康复 → 返回主缸
```
