# 第13章 氨氮循环与微生物系统

> 理解看不见的生命活动——海水缸的核心生态系统

## 13.1 氨氮循环概述

### 13.1.1 什么是氨氮循环？

氨氮循环（Nitrogen Cycle）是海水水族箱中最重要的生物化学过程，它决定了水质的好坏和生物的存活。

**简单比喻：**
```
鱼虾排泄 → 有毒废物 → 细菌分解 → 无毒物质 → 水质安全
```

### 13.1.2 完整的氨氮循环路径

```
┌─────────────────────────────────────────────────────────────────────────┐
│                         氨氮循环示意图                                   │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   有机废物          氨化作用          亚硝化作用         硝化作用          │
│   (食物残渣    ──→   氨(NH₃)   ──→  亚硝酸盐(NO₂) ──→  硝酸盐(NO₃)   │
│    排泄物)           ↓                  ↓                  ↓            │
│                    ↓                  ↓                  ↓            │
│                   有毒              有毒              无毒             │
│                                                                         │
│                                         反硝化作用                      │
│                                         ↓                               │
│                                        氮气(N₂) ↑                       │
│                                        (无害气体逸出)                    │
└─────────────────────────────────────────────────────────────────────────┘
```

### 13.1.3 各阶段详解

| 阶段 | 产物 | 毒性 | 细菌类型 |
|------|------|------|----------|
| 氨化 | 氨 NH₃ | 极高 | 氨化细菌 |
| 亚硝化 | 亚硝酸盐 NO₂ | 高 | 亚硝化细菌 (Nitrosomonas) |
| 硝化 | 硝酸盐 NO₃ | 低 | 硝化细菌 (Nitrobacter) |
| 反硝化 | 氮气 N₂ | 无 | 反硝化细菌 |

## 13.2 核心微生物详解

### 13.2.1 氨化细菌 (Ammonifying Bacteria)

**作用：** 分解有机废物，产生氨

**反应式：**
```
蛋白质 + 水 → 氨 + 其他产物
```

**分布：** 广泛分布于底砂、活石、滤材表面

**注意事项：**
- 任何有机物分解都会产生氨
- 氨对鱼虾有剧毒，必须及时处理

### 13.2.2 亚硝化细菌 (Nitrosomonas)

**作用：** 将氨转化为亚硝酸盐

**反应式：**
```
2NH₃ + 3O₂ → 2HNO₂ + 2H₂O + 能量
氨 + 氧气 → 亚硝酸盐 + 水 + 能量
```

**特点：**
- 生长速度：较慢（繁殖周期约10-20小时）
- 最佳温度：25-30℃
- 最佳 pH：7.5-8.5
- 需氧量：喜欢氧气（好氧菌）

**分布位置：**
- 活石表面
- 底砂上层
- 滤材表面

### 13.2.3 硝化细菌 (Nitrobacter)

**作用：** 将亚硝酸盐转化为硝酸盐

**反应式：**
```
2HNO₂ + O₂ → 2HNO₃ + 能量
亚硝酸盐 + 氧气 → 硝酸盐 + 能量
```

**特点：**
- 生长速度：较慢（繁殖周期约20-30小时）
- 最佳温度：25-30℃
- 最佳 pH：7.5-8.5
- 需氧量：好氧菌

**与亚硝化细菌的关系：**
- 两者互相配合，形成完整的硝化链
- 任何一方不足都会导致中间产物积累

### 13.2.4 反硝化细菌 (Denitrifying Bacteria)

**作用：** 将硝酸盐转化为氮气

**反应式：**
```
HNO₃ → HNO₂ → NO → N₂O → N₂↑
硝酸盐 → 亚硝酸盐 → 一氧化氮 → 氧化亚氮 → 氮气
```

**特点：**
- **厌氧菌**：需要在缺氧环境下工作
- 最佳温度：25-30℃
- 需要碳源（有机物）作为能量来源

**分布位置：**
- 活石内部
- 厚沙底层 (DSB)
- 底缸深滤材区
- 专用反硝化罐

## 13.3 微生物的分布与培养

### 13.3.1 活石中的微生物

**每公斤活石中的微生物含量：**

| 微生物类型 | 数量 (CFU/g) | 作用 |
|-----------|--------------|------|
| 硝化细菌 | 10⁸ - 10⁹ | 将氨转化为硝酸盐 |
| 反硝化细菌 | 10⁶ - 10⁷ | 将硝酸盐转化为氮气 |
| 氨化细菌 | 10⁸ - 10⁹ | 分解有机物产生氨 |
| 其他微生物 | 10⁵ - 10⁶ | 分解有机物 |

### 13.3.2 硝化系统的建立

**自然建立过程：**

| 阶段 | 时间 | 现象 |
|------|------|------|
| 第1周 | 1-7天 | 氨升高，可能死鱼 |
| 第2周 | 8-14天 | 亚硝酸盐升高 |
| 第3-4周 | 15-30天 | 硝酸盐开始积累 |
| 第4周后 | 30天+ | 系统稳定 |

**加速建立的方法：**

1. **添加硝化细菌制剂**
   - Dr. Tim's One and Only
   - Fritz Zyme 7
   - Seachem Stability

2. **引入活石/滤材**
   - 从成熟缸借用的活石
   - 使用旧的滤材

3. **提高温度**
   - 28-30℃ 可加速细菌生长

4. **避免干扰**
   - 不要换水太频繁
   - 不要清洗滤材
   - 不要下药

### 13.3.3 影响硝化效率的因素

| 因素 | 最佳范围 | 影响 |
|------|----------|------|
| 温度 | 25-30℃ | 低于15℃停止活动 |
| pH | 7.5-8.5 | 低于6.5抑制生长 |
| 盐度 | 1.020-1.026 | 过高过低都不利 |
| 溶氧 | >5 ppm | 缺氧会死亡 |
| 光照 | 避免直射 | 紫外线伤害 |

## 13.4 常见问题与解决方案

### 13.4.1 硝化系统崩溃

**原因：**

| 原因 | 说明 |
|------|------|
| 用药 | 抗生素、杀菌剂杀死硝化细菌 |
| 换水 | 大量换水改变水温/盐度 |
| 清洗滤材 | 用自来水冲洗杀死细菌 |
| 长时间停电 | 氧气泵停止，硝化菌死亡 |
| 氨中毒 | 氨浓度过高直接毒死细菌 |

**症状：**
- 氨或亚硝酸盐突然升高
- 水发白/雾蒙蒙
- 鱼虾死亡

**急救步骤：**

1. **立即换水 30-50%**，稀释毒素
2. **停止喂食**，减少氨的产生
3. **增加曝气**，保证溶氧
4. **添加硝化细菌制剂**
5. **如果可能**，引入成熟滤材

### 13.4.2 硝化系统建立太慢

**原因：**

| 原因 | 说明 |
|------|------|
| 温度太低 | 低于20℃细菌不活跃 |
| 滤材太少 | 细菌没有附着场所 |
| 活石不足 | 硝化菌数量不够 |
| 水质不稳 | pH/盐度波动 |

**解决方案：**

1. 提高温度到 28-30℃
2. 增加活石或生物滤材
3. 添加硝化细菌制剂
4. 保持水质稳定
5. 耐心等待（至少4-6周）

### 13.4.3 异养菌爆发

**什么是异养菌？**
- 以有机物为食的细菌
- 繁殖速度极快
- 会导致"水发白"

**原因：**
- 有机物过多（残食、粪便）
- 喂食太多
- 蛋分效率低

**处理方法：**
1. 减少喂食
2. 换水 + 清洁蛋分
3. 使用活性炭
4. 增加生物滤材

## 13.5 微生物与缸类型的关系

### 13.5.1 FOT 纯鱼缸

**特点：**
- 没有活石，硝化能力较弱
- 依赖蛋分和滤材

**建议：**
- 使用厚沙系统 (DSB) 增加反硝化
- 或添加硝化细菌罐
- 定期检测 NO₂

### 13.5.2 FOWLR 活石鱼缸

**特点：**
- 活石提供强大的硝化和反硝化能力
- 微生物群落丰富

**优势：**
- 水质更稳定
- NO₃ 更容易控制

### 13.5.3 LPS 软珊瑚缸

**特点：**
- 需要中等营养盐
- 活石 + 蛋分基本够用

**注意：**
- 避免反硝化过强（NO₃ 太低）
- 定期换水补充微量元素

### 13.5.4 SPS 硬骨珊瑚缸

**特点：**
- 需要极低营养盐
- NP 平衡非常关键
- 需要精确控制微生物群落

**特殊要求：**
- 强大的蛋分
- 适量的反硝化
- 定期检测 NP 值

## 13.6 日常维护要点

### 13.6.1 保护硝化系统

| 应该做 | 不应该做 |
|--------|----------|
| 用原缸水清洗滤材 | 用自来水冲洗滤材 |
| 定期换水 10-20% | 大量换水 (>50%) |
| 慢慢升降温 | 温度剧烈波动 |
| 下药前咨询 | 随意使用抗生素 |

### 13.6.2 监测指标

| 指标 | 正常范围 | 检测频率 |
|------|----------|----------|
| NH₃ | 0 ppm | 每周 |
| NO₂ | 0 ppm | 每周 |
| NO₃ | <20 ppm | 每周 |
| pH | 8.1-8.4 | 每天 |
| 温度 | 25-28℃ | 每天 |

## 13.7 总结

**核心要点：**

1. **氨氮循环是生命线** - 理解它才能养好海水缸
2. **硝化细菌** - 将有毒的氨和亚硝酸盐转化为低毒的硝酸盐
3. **反硝化细菌** - 将硝酸盐转化为无害的氮气
4. **保护硝化系统** - 避免药物、大量换水等破坏
5. **不同缸类型** - 需要不同的微生物管理策略

**记住：**
- 看不见的微生物决定了看得见的生物健康
- 硝化系统的建立需要时间和耐心
- 一旦崩溃，恢复需要几周时间
