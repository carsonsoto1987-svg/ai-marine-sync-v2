# 第12章 常见问题排查

## 12.1 珊瑚问题

### 12.1.1 珊瑚不开

**可能原因：**

| 原因 | 表现 | 解决方法 |
|------|------|----------|
| 盐度不对 | 不舒展 | 调整到1.021-1.025 |
| 温度不对 | 收缩 | 调整到25-26℃ |
| 水质差 | 状态不好 | 检查NO₃、PO₄ |
| 光照不足 | 颜色变淡 | 增加光照 |
| 水流不够 | 不振作 | 增加水流 |
| 适应期 | 新入缸 | 耐心等待 |

**排查顺序：**

```
盐度 → 温度 → 水质 → 光照 → 水流 → 适应
```

### 12.1.2 珊瑚脱骨/死亡

**可能原因：**

- 水质参数剧烈波动
- PO₄过高
- 光照过强
- 温度波动
- 被其他珊瑚攻击

**预防方法：**

- 保持水质稳定
- 控制营养盐
- 合理摆放（保持间距）

## 12.2 水质问题

### 12.2.1 水发白/雾蒙蒙

**原因：** 硝化系统未建立

**解决方法：**

1. 等待硝化系统建立（1-2周）
2. 减少换水（会延长建立时间）
3. 检查滤材是否正常
4. 避免使用药物杀菌

### 12.2.2 水发黄/发绿

**原因：** 有机物过多/藻类

**解决方法：**

- 换水+清洁蛋分
- 使用活性炭
- 控制喂食
- 减少光照

### 12.2.3 NO₃过高

**原因：**

- 换水不足
- 喂食过多
- 蛋分效率低
- 活石/滤材太少

**解决方法：**

- 增加换水频率和量
- 减少喂食
- 清洁/提高蛋分
- 增加活石

### 12.2.4 PO₄过高

**原因：**

- 使用自来水（含磷）
- 喂食过多
- 死鱼未清理

**解决方法：**

- 使用RO/DI水
- 控制喂食
- 及时清理死鱼
- 使用磷酸盐吸附剂

### 12.2.5 FOT缸亚硝酸盐(NO₂)偏高 ⭐ 重点

**问题特征：**
- 测试显示 NO₂ > 0 ppm（理想应为 0）
- 鱼呼吸急促、浮头
- 鱼体表发黑、活力下降

**原因分析：**

| 原因 | 说明 |
|------|------|
| 硝化系统未建立 | 新缸开缸时间太短 |
| 活石/滤材不足 | 缺乏硝化细菌附着场所 |
| 生物负载过高 | 鱼太多或喂食太多 |
| 药物使用 | 抗生素杀死了硝化细菌 |
| 活石死亡 | 活石中的细菌大量死亡 |

**解决方案：**

1. **紧急处理：**
   - 换水 30-50%，降低 NO₂ 浓度
   - 添加硝化细菌制剂（如 Dr. Tim's One and Only）
   - 减少喂食或停止喂食 2-3 天

2. **根本解决：**
   - 增加活石或生物滤材
   - 降低饲养密度
   - 检查蛋分是否正常工作
   - 避免使用抗生素类药物

3. **FOT缸特别建议：**
   - FOT 缸没有活石，硝化能力较弱
   - 建议使用**厚沙系统**（DSB）增加反硝化能力
   - 或添加**硝化细菌罐**（生物球/陶瓷环）

**预防措施：**
- 新缸养水至少 4-6 周再放鱼
- 分批入鱼，不要一次性加太多
- 定期检测 NO₂

### 12.2.6 SPS缸NP偏低 ⭐ 重点

**问题特征：**
- NO₃ < 0.1 ppm，甚至测不出
- PO₄ < 0.01 ppm，甚至测不出
- SPS 颜色变深、变褐（棕褐色）
- SPS 底部白化、顶端生长缓慢

**原因分析：**

| 原因 | 说明 |
|------|------|
| 过度使用吸附剂 | 磷酸盐吸附剂用量过大 |
| 蛋分效率过高 | 大型蛋分处理能力太强 |
| 碳源添加过量 | 乙醇、伏特加等添加太多 |
| 活石过多 | 反硝化能力太强 |
| 喂食太少 | 营养输入不足 |

**NP过低对 SPS 的影响：**
- NO₃ 过低 → 珊瑚缺乏氮源，颜色变深
- PO₄ 过低 → 珊瑚缺乏磷源，生长缓慢
- NP 双低 → 珊瑚"饿死"，底部脱骨

**解决方案：**

1. **提升 NO₃：**
   - 增加喂食量
   - 暂停使用硝酸盐吸附剂
   - 添加氮源（如硝酸钾 KNO₃，需谨慎用量）
   - 减少蛋分运行时间

2. **提升 PO₄：**
   - 增加喂食量（特别是肉食性饲料）
   - 暂停磷酸盐吸附剂
   - 使用 " Reef Roids " 等珊瑚粮

3. **SPS 缸理想 NP 范围：**

| 参数 | 理想范围 | 说明 |
|------|----------|------|
| NO₃ | 1-5 ppm | 不能太高，也不能太低 |
| PO₄ | 0.01-0.03 ppm | 同样需要维持 |

4. **监控方法：**
   - 使用高精度测试剂（如 Hanna 测试仪）
   - 每周检测 NP 值
   - 建立 NP 变化日志

**特别提醒：**
- SPS 缸不是 NP 越低越好！
- NP 需要维持在一定水平，珊瑚才能正常生长和显色
- "超低营养盐系统"（ULNS）需要专业知识，新手不推荐

## 12.3 藻类问题

### 12.3.1 藻类爆发

**原因：**

- 光照过强
- 营养盐过高
- 换水不足
- 蛋分效率低

**解决方法：**

| 程度 | 方法 |
|------|------|
| 轻度 | 增加换水、减少光照 |
| 中度 | 物理清除+除藻生物 |
| 重度 | 彻底翻缸 |

### 12.3.2 绿毛藻

**解决方法：**

- 引入蓝吊、黄金吊
- 减少光照时间
- 控制喂食
- 人工清除

## 12.4 鱼类问题

### 12.4.1 鱼状态不好

**可能原因：**

- 水质问题
- 疾病
- 打斗
- 不适应

**排查步骤：**

1. 测试水质
2. 观察身体有无异常
3. 观察与其他鱼的关系
4. 检查喂食情况

### 12.4.2 鱼打斗

**原因：** 领地意识

**解决方法：**

- 增加躲避物
- 重新摆放造景
- 移除攻击性强的鱼
- 暂时隔离

## 12.5 设备问题

### 12.5.1 蛋分不出泡沫

**原因：**

- 气泡石堵塞
- 进气量不对
- 水位不对

**解决方法：**

- 清洗气泡石
- 调节进气阀
- 调整水位

### 12.5.2 加热棒不工作

**解决方法：**

- 检查电源
- 检查温控设置
- 更换加热棒（如损坏）

### 12.5.3 造浪泵不转

**解决方法：**

- 检查电源
- 清洗轴承
- 更换电机
