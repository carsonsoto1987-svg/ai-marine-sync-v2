# 第11章 常见疾病与治疗

## 11.1 疾病预防

### 11.1.1 预防的重要性

海水鱼一旦生病，治疗难度较大，预防比治疗更重要。

### 11.1.2 预防措施

1. **良好的水质**：最关键的预防措施
2. **正确的过水**：避免带入病原
3. **稳定的参数**：避免剧烈波动
4. **合理的喂食**：营养均衡，不过量
5. **新鱼检疫**：隔离观察后再入缸
6. **定期观察**：及早发现问题

## 11.2 常见疾病

### 11.2.1 白点病

**最常见的海水鱼疾病**

**病原：** 刺激隐核虫（Cryptocaryon irritans）

**症状：**

- 体表出现白色小点（像盐粒）
- 频繁蹭缸、擦石
- 食欲减退
- 游动迟缓
- 呼吸急促

**传染性：** 非常强，会传染给其他鱼

**治疗方案：**

| 方法 | 操作 | 注意事项 |
|------|------|----------|
| 铜药治疗 | 隔离缸加铜药，浓度0.15-0.2mg/L | 2-4周，结束后观察4周 |
| 升温疗法 | 水温升至28-30℃ | 持续2-3周 |
| 渗透压疗法 | 盐度降至1.008-1.009 | 持续4周后慢慢恢复 |

**注意事项：**

- 铜药对珊瑚有毒，不能用于主缸
- 治疗期间保持水质稳定
- 治疗后要换水稀释

### 11.2.2 烂尾/烂鳍病

**病原：** 细菌（柱状黄杆菌等）或水质恶化

**症状：**

- 尾鳍或背鳍发白、溃烂
- 鳍条边缘破损
- 严重时鳍条断裂
- 游动困难

**治疗：**

1. 隔离治疗
2. 加入抗生素（新霉素、氯霉素等）
   - 剂量：每升水50mg
   - 疗程：连续7天
3. 改善水质
4. 盐浴：1.3克/升，每日30分钟

### 11.2.3 肠炎

**病原：** 细菌感染

**症状：**

- 拉白便
- 肛门红肿
- 食欲不振
- 离群独处

**治疗：**

1. 停食2天
2. 下土霉素或氟哌酸
3. 保持水质清洁
4. 恢复后逐步增加喂食

### 11.2.4 水霉病

**病原：** 真菌（绵霉属等）

**症状：**

- 白色棉絮状菌丝
- 通常附着在伤口处
- 严重时覆盖全身
- 鱼体消瘦

**治疗：**

1. 升温1-2℃
2. 盐浴：每升水3克盐，每日30分钟
3. 黄粉药浴
4. 改善水质

### 11.2.5 打印病（腐皮病）

**病原：** 细菌（荧光假单胞菌）

**症状：**

- 皮肤出现溃烂病灶
- 病灶处鳞片脱落
- 呈鲜红色或灰白色
- 严重时危及生命

**治疗：**

1. 隔离治疗
2. 抗生素药浴
3. 消毒缸体

### 11.2.6 纤毛虫病

**病原：** 纤毛虫（如车轮虫）

**症状：**

- 频繁蹭缸
- 体表有白色黏膜
- 呼吸急促
- 食欲不振

**治疗：**

1. 淡水浴：3-5分钟
2. 铜药治疗
3. 保持水质清洁

## 11.3 用药注意事项

### 11.3.1 用药原则

1. **确诊后再用药**：不要盲目用药
2. **隔离治疗**：在隔离缸中治疗
3. **足量足疗程**：不要中途停药
4. **观察反应**：注意副作用

### 11.3.2 常用药物

| 药物 | 用途 | 注意事项 |
|------|------|----------|
| 铜药 | 白点病、纤毛虫 | 对珊瑚有毒，严格控制浓度 |
| 黄粉 | 细菌感染 | 注意增氧 |
| 土霉素 | 细菌感染 | 可能影响硝化系统 |
| 甲基蓝 | 预防真菌、细菌 | 会染色 |
| 亚甲基蓝 | 白点病预防 | 对鱼有毒性 |

### 11.3.3 常见问题

- **用药过量**：会导致鱼死亡
- **用药不足**：无法治愈
- **药物残留**：对珊瑚和其他生物有害

## 11.4 疾病诊断流程

```
观察症状 → 初步判断 → 隔离病鱼 → 对症治疗 → 观察效果 → 治愈后入缸
```
