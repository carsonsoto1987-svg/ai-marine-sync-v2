# 第7章 日常维护

## 7.1 维护的重要性

海水缸是一个动态的生态系统，需要持续的维护才能保持稳定。良好的维护不仅能保证生物的健康，还能延长设备寿命，降低运营成本。

**维护的核心目标：**

1. 维持水质稳定
2. 预防疾病发生
3. 保持缸体美观
4. 延长设备寿命

## 7.2 定期维护操作

### 7.2.1 每日操作

| 操作 | 时间 | 耗时 | 说明 |
|------|------|------|------|
| 检查温度 | 5秒 | 必须 |
| 观察生物状态 | 2分钟 | 推荐 |
| 清理蛋分收集杯 | 1分钟 | 必须 |
| 喂食 | 5分钟 | 必须 |

**每日观察要点：**

- 鱼是否活跃，有无异常行为
- 珊瑚是否舒展
- 有无死亡生物
- 水面有无异常泡沫

### 7.2.2 每周操作

| 操作 | 时间 | 频率 | 说明 |
|------|------|------|------|
| 测试水质 | 15分钟 | 1-2次 | 盐度、pH、NO₃ |
| 换水 | 30分钟 | 20-25% | 最重要的工作 |
| 清洗滤棉 | 10分钟 | 1次 | 物理过滤维护 |
| 缸壁清洁 | 15分钟 | 1次 | 保持透明 |
| 检查设备 | 10分钟 | 1次 | 造浪、加热等 |

### 7.2.3 每月操作

| 操作 | 频率 | 说明 |
|------|------|------|
| 活石刷洗 | 1-2次 | 去除藻类 |
| 深度清洁滤材 | 1次 | 切勿用自来水 |
| 检查RO/DI设备 | 1次 | 更换滤芯 |
| 检查管路 | 1次 | 有无泄漏 |
| 设备维护 | 1次 | 清洁和校准 |

### 7.2.4 季度/年度操作

| 操作 | 频率 | 说明 |
|------|------|------|
| 更换滤芯 | 季度 | RO/DI设备 |
| 深度清洁底缸 | 半年 | 清除沉积物 |
| 设备检修 | 一年 | 检查老化 |
| 活石补充 | 按需 | 补充或更换 |

## 7.3 换水

### 7.3.1 重要性

换水是海水缸维护中最重要的工作：

1. **去除积累的废物**：NO₃、PO₄等
2. **补充微量元素**：Ca、Mg、K等
3. **维持水质稳定**：保持各项参数
4. **刺激生物**：新水可以刺激生物活性

### 7.3.2 换水量

| 缸类型 | 建议换水频率 | 换水比例 |
|--------|--------------|----------|
| FOT | 每周 | 20-25% |
| LPS | 每周 | 20-25% |
| SPS | 每周 | 25-30% |
| 新开缸 | 初期 | 每日10% |

### 7.3.3 换水步骤

1. **准备新海水**：
   - 使用RO/DI水
   - 按比例加入海盐
   - 充分搅拌并测试盐度
   - 调温至与缸内一致

2. **抽水**：
   - 使用虹吸管抽水
   - 抽至距底约5cm
   - 避免吸入活石

3. **添加新水**：
   - 缓慢倒入新水
   - 避免直接冲刷生物

4. **测试水质**：
   - 测试新水盐度
   - 确认各项参数正常

### 7.3.4 注意事项

- 永远使用RO/DI水
- 新水温度要与缸内一致
- 换水后测试水质
- 换水前停止喂食

## 7.4 蛋分维护

### 7.4.1 日常维护

1. **每日清理收集杯**：
   - 倒掉泡沫和废水
   - 用海水冲洗收集杯
   - 检查收集情况

2. **检查气泡产生**：
   - 观察气泡是否正常
   - 检查水位调节
   - 确认无泄漏

### 7.4.2 定期维护

| 项目 | 频率 | 方法 |
|------|------|------|
| 清洗喷嘴 | 每周 | 海水冲洗 |
| 清洗气泡石 | 每月 | 浸泡清洗 |
| 检查气泵 | 每月 | 听声音是否正常 |
| 深度清洁 | 季度 | 分解清洗 |

## 7.5 水质检测

### 7.5.1 检测频率

| 参数 | 基础频率 | 进阶频率 |
|------|----------|----------|
| 盐度 | 每周 | 每次换水 |
| 温度 | 每天 | - |
| pH | 每周 | 每日 |
| NO₃ | 每周 | 每日 |
| KH | 每周 | 每周多次 |
| PO₄ | 每周 | 每日 |
| NH₃/NO₂ | 每月 | 每周 |

### 7.5.2 检测方法

**测试剂：**

1. 取水样
2. 加入试剂
3. 摇匀等待反应
4. 对比色卡读数

**电子测试仪：**

1. 校准仪器
2. 放入水样
3. 读取数据
4. 记录分析

建议建立维护日志：

## 7.6 藻类治理

### 7.6.1 常见藻类

| 藻类 | 特点 | 处理方法 |
|------|------|----------|
| 褐藻 | 最早出现，褐色 | 换水+光照 |
| 绿藻 | 营养盐高，绿色 | 控制喂食+换水 |
| 绿毛藻 | 丝状，绿色 | 生物治理+减少光照 |
| 气泡藻 | 气泡状，绿色 | 加强过滤+换水 |
| 蓝藻 | 蓝绿色，黏滑 | 遮光+换水 |

### 7.6.2 预防措施

1. **控制光照**：每日8-10小时，避免直射
2. **控制喂食**：少量多次，5分钟内吃完
3. **保持换水**：定期换水
4. **蛋分工作**：确保蛋分正常
5. **及时清理**：死鱼、残饵及时清除

### 7.6.3 治理方法

**轻度藻类：**

- 增加换水频率
- 减少喂食
- 适当减少光照

**中度藻类：**

- 物理清除（刷子刮）
- 引入除藻生物
- 使用除藻剂

**重度藻类：**

- 彻底翻缸
- 重新开缸
- 查找根本原因

## 7.7 设备维护

### 7.7.1 造浪泵

- 每月清洗外表面
- 季度检查轴承
- 每年更换轴承

### 7.7.2 照明设备

- 每月清洁灯罩
- 每季检查散热
- 每年更换LED灯珠（如有）

### 7.7.3 加热设备

- 季度检查温控准确性
- 每年检查加热元件
- 及时更换老化设备
