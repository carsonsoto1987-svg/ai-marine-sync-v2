# 第9章 开缸流程（柏林系统）

## 9.1 柏林系统概述

柏林系统（Berlin System）是目前最主流的海水缸系统，其核心是利用活石作为生物过滤介质，配合蛋白质分离器来维持水质。

**柏林系统的核心理念：**

```
自然海水生态系统 → 活石 + 蛋分 → 人工海水生态系统
```

## 9.2 开缸准备

### 9.2.1 设备准备

**必需设备：**

| 设备 | 说明 |
|------|------|
| 鱼缸 | 60cm以上推荐 |
| 蛋分 | 处理量2-3倍水体 |
| 造浪泵 | 1-2台 |
| 照明 | LED灯具 |
| 加热棒 | 300W |
| 底缸/过滤 | 具备物理/生物过滤 |
| 活石 | 0.5-0.8kg/L |

**可选设备：**

- RO/DI机
- 冷水机（高温地区）
-  calcium reactor（钙反）
- 自动喂食器

### 9.2.2 水族箱消毒

**新缸消毒：**

1. 用海水专用消毒剂
2. 或高锰酸钾（稀释）
3. 浸泡24小时后冲洗干净

**二手缸消毒：**

1. 彻底清洗
2. 用消毒剂浸泡
3. 暴晒或紫外线消毒

## 9.3 开缸步骤

### 步骤1：铺底砂与活石

**底砂铺设：**

- 厚度：5-10cm
- 粒径：2-4mm粗砂
- 分布：均匀铺设

**活石摆放：**

- 原则：美观自然，留水流通道
- 堆叠：不要堆太高，避免倒塌
- 密度：0.5-0.8kg/L
- 注意：活石孔隙要多

### 步骤2：配制海水

**海盐配制：**

- 用RO/DI水
- 按33g/L的比例加入海盐
- 充分搅拌溶解

**测试比重：**

- 标准：1.021-1.026
- 温度校正后确认

### 步骤3：设备运行

**开启设备：**

1. 开启过滤循环
2. 开启蛋分（调整到适当进气量）
3. 开启造浪泵
4. 开启加热棒（设置26℃）

**初始设置：**

- 蛋分：调整到正常出泡
- 造浪：中低速，避免直接吹活石
- 照明：先不开启或低强度

### 步骤4：爆藻阶段（关键）

**什么是爆藻：**

爆藻是活石中的营养盐和藻类在强光下爆发生长的阶段，是建立生态系统的正常过程。

**爆藻过程：**

```
第1周：开始有褐藻生长
第2周：褐藻茂盛，可能覆盖活石
第3周：藻类开始变白（白化）
第4周：白化藻类死亡脱落
```

**爆藻管理：**

| 时间 | 操作 | 说明 |
|------|------|------|
| 1-2周 | 强光照射 | 每天8-12小时 |
| 2-3周 | 观察 | 等待藻类白化 |
| 3-4周 | 刷藻 | 刷掉白化藻类 |
| 期间 | 换水 | 每周换水20-30% |

**注意事项：**

- 爆藻期间不要放入生物
- 蛋分要正常工作
- 避免阳光直射（会爆藻太猛）

### 步骤5：刷藻清理

**时机：** 藻类大部分白化后

**操作：**

1. 用软刷轻轻刷掉活石上的白化藻
2. 用吸管吸出底部的藻类残渣
3. 换水20-30%

**可能需要多次：** 重复刷藻换水过程2-3次

### 步骤6：养水检测

**检测项目：**

| 参数 | 标准 | 说明 |
|------|------|------|
| 盐度 | 1.021-1.026 | 正常 |
| 温度 | 25-26℃ | 正常 |
| pH | 8.0-8.3 | 正常 |
| NH₃ | 0 ppm | 正常 |
| NO₂ | 0 ppm | 正常 |
| NO₃ | <20 ppm | 可接受 |

**达标标准：** NH₃和NO₂都为0，NO₃低于20ppm

### 步骤7：闯缸鱼测试

**闯缸鱼选择：**

| 品种 | 特点 | 数量 |
|------|------|------|
| 蓝魔 | 便宜、皮实 | 3-5条 |
| 泗水玫瑰 | 人工繁殖、适应性好 | 3-5条 |
| 小丑鱼 | 入门级 | 2-3条 |

**测试期：**

- 闯缸鱼入缸后观察1-2周
- 如果状态良好，说明水质稳定
- 可以继续添加其他生物

### 步骤8：生物入缸

**添加顺序：**

```
闯缸鱼（1-2周）→ 活石生物（虾蟹螺）→ 温和的小型鱼 → 其他鱼 → 珊瑚
```

**间隔时间：**

- 每批生物间隔1-2周
- 观察水质变化
- 确认无问题后再添加

## 9.3 时间节点汇总

| 阶段 | 时间 | 关键操作 |
|------|------|----------|
| 准备 | 1天 | 设备安装、消毒 |
| 铺装 | 1天 | 活石、底砂摆放 |
| 开循环 | 1天 | 设备开启 |
| 爆藻 | 2-4周 | 等待藻类白化 |
| 清理 | 1-2周 | 刷藻、换水 |
| 养水 | 1-2周 | 检测达标 |
| 闯缸 | 1-2周 | 测试水质 |
| 正式 | - | 逐步添加生物 |

**总计：** 约6-8周

## 9.4 常见问题

### 9.4.1 爆藻太严重

**原因：** 光照太强、活石营养盐太多

**解决：** 减少光照、加大换水

### 9.4.2 水一直雾蒙蒙

**原因：** 硝化系统未建立

**解决：** 等待，爆藻后会变清

### 9.4.3 NO₃降不下来

**原因：** 活石太少、蛋分效率低、喂食过多

**解决：** 增加活石、提高蛋分、减少喂食
