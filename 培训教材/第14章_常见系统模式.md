# 第14章 常见系统模式

> 主流海水缸过滤系统的原理、特点与选择

## 14.1 系统模式概述

海水水族箱的过滤系统经历了多年发展，形成了多种成熟的技术路线。不同的系统模式适合不同的养殖目标和经济预算。

**选择系统的考虑因素：**

| 因素 | 问题 |
|------|------|
| 养殖目标 | 纯鱼？LPS？SPS？ |
| 经验水平 | 新手？老手？ |
| 预算 | 基础？进阶？ |
| 维护时间 | 每周几小时？ |
| 空间 | 桌面？客厅？ |

## 14.2 柏林系统 (Berlin Method)

### 14.2.1 系统原理

柏林系统是最经典、最流行的海水缸过滤系统，由德国柏林的水族爱好者在1970年代创立。

**核心原理：**
```
鱼虾排泄物 → 蛋分去除有机物 → 
活石硝化细菌 → 转化氨 → 亚硝酸盐 → 硝酸盐 →
活石反硝化细菌 → 氮气逸出
```

### 14.2.2 系统组成

| 组件 | 作用 | 重要性 |
|------|------|--------|
| 活石 | 生物过滤、硝化/反硝化 | ⭐⭐⭐⭐⭐ |
| 蛋白分离器 | 去除蛋白质、有机物 | ⭐⭐⭐⭐⭐ |
| 活砂/底砂 | 辅助过滤、微生物栖息 | ⭐⭐⭐ |
| 照明 | 珊瑚光合作用 | ⭐⭐⭐⭐ |
| 造浪泵 | 水流交换 | ⭐⭐⭐⭐ |
| 加热/制冷 | 温度控制 | ⭐⭐⭐ |

### 14.2.3 系统图示

```
┌─────────────────────────────────────────────────────────┐
│                    柏林系统示意图                         │
├─────────────────────────────────────────────────────────┤
│                                                         │
│    ┌─────────┐      ┌─────────┐      ┌─────────┐       │
│    │  主缸   │ ←──→ │  蛋分   │ ←──→ │  底缸   │       │
│    │         │      │         │      │         │       │
│    │  鱼/珊瑚 │      │  去除   │      │  滤材   │       │
│    │         │      │  有机物 │      │  活石   │       │
│    └─────────┘      └─────────┘      └─────────┘       │
│         ↑                                      │        │
│         └──────────────────────────────────────┘        │
│                      水流循环                            │
└─────────────────────────────────────────────────────────┘
```

### 14.2.4 优缺点

**优点：**

1. 系统成熟可靠
2. 活石造景美观
3. 微生物群落丰富
4. 适合各种生物
5. 维护相对简单

**缺点：**

1. 活石成本较高
2. 需要定期维护
3. 反硝化能力有限
4. 可能破坏海洋生态（活石采集）

### 14.2.5 适合目标

| 缸类型 | 适合度 |
|--------|--------|
| FOT | ⭐⭐⭐ |
| FOWLR | ⭐⭐⭐⭐⭐ |
| LPS | ⭐⭐⭐⭐⭐ |
| SPS | ⭐⭐⭐ |
| 混合型 | ⭐⭐⭐⭐ |

### 14.2.6 建造要点

1. **活石用量：** 0.5-0.8 kg/L 水体
2. **蛋分选择：** 处理量 2-3 倍水体
3. **养水时间：** 至少 4-6 周
4. **换水频率：** 每周 10-20%

## 14.3 Zeovit 系统

### 14.3.1 系统原理

Zeovit（ZEO）系统是一种源自德国的进阶珊瑚养殖系统，由 Zeovit 公司推广。该系统强调使用极低营养盐（ULNS）来养殖 SPS 珊瑚。

**核心原理：**
```
强蛋分 → 去除大量营养盐 →
ZEOvit 吸附剂 → 进一步去除 PO₄ →
ZEOfish/ZEOstart → 提供珊瑚所需微量元素 →
SPS 珊瑚最佳状态
```

### 14.3.2 系统组成

| 组件 | 作用 | 重要性 |
|------|------|--------|
| 超强蛋分 | 大量去除有机物 | ⭐⭐⭐⭐⭐ |
| 吸附剂 | 去除 PO₄ | ⭐⭐⭐⭐⭐ |
| 沸石 | 吸附氨等 | ⭐⭐⭐⭐ |
| 微量添加剂 | 补充微量元素 | ⭐⭐⭐⭐ |
| 活石/滤材 | 基础过滤 | ⭐⭐⭐ |

### 14.3.3 Zeovit 产品系列

| 产品 | 作用 | 用法 |
|------|------|------|
| ZEOstart | 碳源/营养 | 每天添加 |
| ZEOfish | 微生物营养 | 每天添加 |
| ZEObasis | 珊瑚基础营养 | 每天添加 |
| Zeovit | 沸石滤材 | 放入滤袋 |
| PH-Ex | 磷酸盐吸附剂 | 放入滤袋 |

### 14.3.4 优缺点

**优点：**

1. SPS 珊瑚状态极佳
2. 颜色鲜艳
3. 生长快速
4. 系统化程度高

**缺点：**

1. 成本高昂
2. 操作复杂
3. NP 容易失控
4. 需要精确控制

### 14.3.5 适合目标

| 缸类型 | 适合度 |
|--------|--------|
| FOT | ⭐ |
| FOWLR | ⭐ |
| LPS | ⭐⭐⭐ |
| SPS | ⭐⭐⭐⭐⭐ |
| 混合型 | ⭐⭐⭐ |

### 14.3.6 风险提示

⚠️ **Zeovit 系统风险：**

1. **NP 过低风险**
   - 珊瑚可能"饿死"
   - 需要精确监控

2. **吸附剂饱和**
   - 需要定期更换
   - 饱和后可能释放

3. **成本高昂**
   - 吸附剂消耗大
   - 添加剂价格高

4. **不适合新手**
   - 需要丰富经验
   - 容错率低

## 14.4 DSB 系统 (Deep Sand Bed)

### 14.4.1 系统原理

DSB（Deep Sand Bed，厚沙系统）利用厚沙层中的厌氧反硝化细菌来去除硝酸盐。

**核心原理：**
```
有机物 → 沉底 → 表层好氧菌分解 →
进入中层 → 厌氧反硝化 →
底层 → 氮气逸出
```

### 14.4.2 沙层结构

```
┌─────────────────────────────────────┐
│  DSB 沙层结构                        │
├─────────────────────────────────────┤
│                                     │
│  表层 (0-2cm) - 好氧区              │
│  ┌─────────────────────────────┐   │
│  │  硝化细菌 | 分解者           │   │
│  ├─────────────────────────────┤   │
│  │                              │   │
│  中层 (2-8cm) - 兼氧区          │
│  ┌─────────────────────────────┐   │
│  │  反硝化细菌开始活跃           │   │
│  ├─────────────────────────────┤   │
│  │                              │   │
│  底层 (8-15cm+) - 厌氧区       │
│  ┌─────────────────────────────┐   │
│  │  厌氧反硝化细菌              │   │
│  │  产生氮气 ↑                  │   │
│  └─────────────────────────────┘   │
│                                     │
│  底部 → 无氧，黑色 (硫化氢)        │
└─────────────────────────────────────┘
```

### 14.4.3 系统参数

| 参数 | 建议值 |
|------|--------|
| 沙层厚度 | 10-15 cm |
| 沙子粒径 | #2-4 (1-2mm) |
| 沙子类型 | 阿拉伯钙砂/ Fiji 砂 |
| 底面积 | 越大越好 |

### 14.4.4 优缺点

**优点：**

1. 自然反硝化
2. 减少换水频率
3. 成本较低
4. 系统稳定
5. 适合 FOT/FOWLR

**缺点：**

1. 需要较厚底砂
2. 可能产生硫化氢
3. 建立时间长 (3-6个月)
4. 不适合SPS

### 14.4.5 适合目标

| 缸类型 | 适合度 |
|--------|--------|
| FOT | ⭐⭐⭐⭐⭐ |
| FOWLR | ⭐⭐⭐⭐⭐ |
| LPS | ⭐⭐⭐ |
| SPS | ⭐ |
| 混合型 | ⭐⭐⭐ |

### 14.4.6 维护要点

1. **不要翻动底砂** - 会释放硫化氢
2. **定期清洁表面** - 防止藻类
3. **注意硫化氢** - 黑色底层是危险的
4. **缓慢建立** - 需要3-6个月成熟

## 14.5 藻缸系统 (ATS / Algae Tank)

### 14.5.1 系统原理

藻缸系统（ATS = Algae Turf Scrubber）利用大型藻类吸收营养盐来控制 NO₃ 和 PO₄。

**核心原理：**
```
营养盐 (NO₃/PO₄) → 藻类吸收 →
光合作用 →
藻类收割/取出 → 营养盐移除
```

### 14.5.2 系统类型

| 类型 | 说明 |
|------|------|
| 独立藻缸 | 单独的缸体培养藻类 |
| 藻屏 (Scrubber) | 垂直板状藻类生长面 |
| 藻桶 | 容器内培养藻类 |

### 14.5.3 常见藻类

| 藻类 | 特点 |
|------|------|
| 绿藻 (Chaetomorpha) | 生长快，易维护 |
| 葡萄藻 (Caulerpa) | 吸收能力强 |
| 红藻 ( Gracilaria) | 吸收 PO₄ 效果好 |

### 14.5.4 系统图示

```
┌─────────────────────────────────────────────────┐
│                藻缸系统示意图                     │
├─────────────────────────────────────────────────┤
│                                                 │
│    主缸水体 ──→ 泵 ──→ 藻缸 ──→ 回归主缸        │
│                        ↓                        │
│                       藻类                       │
│                       ↓                        │
│                     收割                         │
│                     ↓                        │
│                   营养盐移除                     │
└─────────────────────────────────────────────────┘
```

### 14.5.5 优缺点

**优点：**

1. 自然去除营养盐
2. 减少换水
3. 提供生物饵料
4. 增加生物多样性
5. 成本较低

**缺点：**

1. 需要光照
2. 需要管理藻类生长
3. 可能影响美观
4. 收割需要人工

### 14.5.6 适合目标

| 缸类型 | 适合度 |
|--------|--------|
| FOT | ⭐⭐⭐⭐ |
| FOWLR | ⭐⭐⭐⭐ |
| LPS | ⭐⭐⭐⭐ |
| SPS | ⭐⭐⭐⭐⭐ |
| 混合型 | ⭐⭐⭐⭐ |

## 14.9 总结

**核心要点：**

1. **柏林系统** - 最经典、通用性强
2. **Zeovit** - 高级 SPS 系统，成本高
3. **DSB** - 自然反硝化，适合 FOT
4. **藻缸** - 绿色过滤，成本适中
5. **组合系统** - 取长补短，效果最佳

**记住：**
- 没有最好的系统，只有最适合你的系统
- 新手建议从柏林系统开始
- 经验积累后再尝试进阶系统
- 系统稳定比先进更重要
