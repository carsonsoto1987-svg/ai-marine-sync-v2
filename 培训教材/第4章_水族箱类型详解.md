# 第4章 水族箱类型详解

## 4.1 水族箱类型概述

海水水族箱根据养殖目标和生物配置的不同，可以分为多种类型。不同类型的缸在设备配置、养殖难度、运营成本等方面有显著差异。

**选择缸类型的考虑因素：**

1. **养殖目标**：纯鱼、珊瑚还是混合
2. **经验水平**：新手还是有经验的玩家
3. **预算**：设备投入和后期维护成本
4. **空间**：缸体尺寸和放置位置
5. **时间**：维护时间和精力投入

## 4.2 FOT (Fish Only Tank) 纯鱼缸

### 4.2.1 定义与特点

**定义：** 只养鱼，不养珊瑚和无脊椎动物的海水缸。

**核心理念：** 以鱼为主要观赏对象，不追求珊瑚状态，强调鱼类的展示和互动。

### 4.2.2 适合生物

**推荐鱼类：**

| 类别 | 品种示例 | 特点 |
|------|----------|------|
| 小丑鱼 | 公子小丑、透红小丑 | 互动性好，易饲养 |
| 倒吊类 | 黄金吊、紫吊、大帆吊 | 色彩鲜艳，活泼 |
| 神仙鱼 | 皇后神仙、蓝面神仙 | 体型优雅 |
| 炮弹鱼 | 闊口鱼、灰神仙 | 形态独特 |
| 雷达鱼 | 紫雷达、线鳍雷达 | 体型小巧 |
| 青蛙鱼 | 麒麟青蛙鱼 | 形态特殊 |
| 鳗鱼 | 草鳗、花园鳗 | 观赏性强 |

**不推荐品种：**

- 肉食性强的鱼类（可能攻击其他鱼）
- 领地意识强的鱼类（易发生打斗）
- 对水质要求极高的鱼类

### 4.2.3 设备配置

**基础设备：**

| 设备 | 推荐配置 | 预算 |
|------|----------|------|
| 缸体 | 60-120cm | 500-2000元 |
| 蛋分 | 处理量1-2倍水体 | 300-600元 |
| 过滤 | 底缸或背滤 | 200-500元 |
| 照明 | 普通LED | 200-400元 |
| 造浪 | 1-2台 | 200-400元 |
| 加热棒 | 200-300W | 100-200元 |
| **合计** | - | **1500-4000元** |

### 4.2.4 水质要求

| 参数 | 范围 | 频率 |
|------|------|------|
| 盐度 | 1.020-1.025 | 每周 |
| 温度 | 25-28℃ | 每天 |
| pH | 8.0-8.4 | 每周 |
| NO3 | <50 ppm | 每周 |
| NH3/NO2 | 0 ppm | 每月 |

### 4.2.5 优点与缺点

**优点：**

1. 成本相对较低
2. 难度较低，适合新手
3. 病害较少
4. 鱼的状态容易维持
5. 可以饲养大型鱼类

**缺点：**

1. 不能养珊瑚
2. 观赏性相对单一
3. 无法体验珊瑚的美妙状态

### 4.2.6 适合人群

- 刚入坑的新手
- 预算有限的玩家
- 偏好大型鱼类的爱好者
- 时间有限的上班族

## 4.3 FOWLR (Fish Only With Live Rock) 活石鱼缸

### 4.3.1 定义与特点

**定义：** 养鱼+活石，但不养殖珊瑚。

**核心理念：** 利用活石提供生物过滤，同时造景美观。

### 4.3.2 适合生物

- 各类海水鱼（小丑鱼、吊类、神仙鱼等）
- 活石（提供过滤和造景）
- 清洁生物（虾、螺、蟹等无脊椎动物）

### 4.3.3 设备配置

**基础设备：**

| 设备 | 推荐配置 | 预算 |
|------|----------|------|
| 缸体 | 60-120cm | 500-2000元 |
| 活石 | 0.5-0.8kg/L | 200-1000元 |
| 蛋分 | 处理量1-2倍水体 | 300-600元 |
| 过滤 | 底缸 | 300-800元 |
| 照明 | 普通LED | 200-400元 |
| 造浪 | 1-2台 | 200-400元 |
| 加热棒 | 200-300W | 100-200元 |
| **合计** | - | **1800-5500元** |

### 4.3.4 优缺点

**优点：**

1. 生物过滤效果好
2. 活石造景美观
3. 维持成本适中
4. 可以观察活石上的生物

**缺点：**

1. 活石成本较高
2. 需要定期维护活石
3. 采集活石可能破坏海洋生态

### 4.3.5 注意事项

1. **选择可持续采购的活石**
2. **避免使用不明来源的活石**
3. **注意活石的重量和摆放安全**

## 4.4 LPS (Large Polyped Stony) 软珊瑚缸

### 4.4.1 定义与特点

**定义：** 饲养大水螅体软珊瑚的珊瑚缸，是最常见的珊瑚缸类型。

**核心理念：** 在保证珊瑚状态的前提下养鱼，追求色彩丰富的视觉效果。

### 4.4.2 适合生物

**LPS软珊瑚：**

| 品种 | 特点 | 价格 |
|------|------|------|
| 纽扣珊瑚 | 最常见，新手首选 | 10-100元 |
| 皮革珊瑚 | 质感独特，容易养 | 30-200元 |
| 榔头珊瑚 | 形态优美 | 50-300元 |
| 火柴头珊瑚 | 形态独特 | 50-250元 |
| 蛙卵珊瑚 | 圆形可爱 | 100-500元 |
| 炮仗花 | 形态特殊 | 30-200元 |
| 太阳花 | 颜色鲜艳 | 20-150元 |
| 宝石花 | 色彩丰富 | 30-200元 |
| 麦穗 | 小巧可爱 | 20-100元 |

**适合混养的鱼类：**

- 小丑鱼（与珊瑚共生）
- 海金鱼（活跃可爱）
- 蓝魔鱼（体型小）
- 泗水玫瑰（人工繁殖，适应性好）
- 雷达鱼（体型小）

**适合混养的无脊椎动物：**

- 清洁虾
- 医生虾
- 翻沙螺
- 寄居蟹

### 4.4.3 环境要求

**光照要求：**

| 参数 | 范围 | 说明 |
|------|------|------|
| PAR | 50-150 | 中低光 |
| 光谱 | 全光谱或蓝白 | 模拟自然光 |
| 每天时长 | 8-10小时 | 避免过长 |

**水流要求：**

| 参数 | 范围 | 说明 |
|------|------|------|
| 强度 | 中低 | 避免直冲 |
| 方向 | 多角度 | 模拟自然洋流 |
| 方式 | 造浪泵 | 每日变换方向 |

**水质要求：**

| 参数 | 范围 | 说明 |
|------|------|------|
| NO3 | <10 ppm | 越低越好 |
| PO4 | <0.1 ppm | 越低越好 |
| KH | 7-9 dKH | 维持pH稳定 |
| pH | 8.1-8.3 | 保持稳定 |

### 4.4.4 设备配置

**基础设备：**

| 设备 | 推荐配置 | 预算 |
|------|----------|------|
| 缸体 | 60-120cm | 500-2000元 |
| 活石 | 0.5-0.8kg/L | 200-1000元 |
| 蛋分 | 处理量2-3倍水体 | 400-800元 |
| 照明 | 全光谱LED | 500-1000元 |
| 造浪 | 2台以上 | 300-600元 |
| 底缸 | 带滤材区 | 500-1000元 |
| 加热棒 | 300W | 150-250元 |
| **合计** | - | **2500-6500元** |

### 4.4.5 优点与缺点

**优点：**

1. 色彩丰富
2. 相对容易饲养
3. 价格适中（从几十元到几百元不等）
4. 对设备要求不如SPS苛刻
5. 可选择的品种多

**缺点：**

1. 部分品种对水质敏感
2. 部分品种会咬其他珊瑚
3. 需要一定的光照和水流管理

### 4.4.6 适合人群

- 有基础的玩家
- 喜欢色彩丰富的玩家
- 想要尝试珊瑚的新手
- 预算中等的玩家

## 4.5 SPS (Small Polyped Stony) 硬骨珊瑚缸

### 4.5.1 定义与特点

**定义：** 饲养小水螅体硬骨珊瑚的珊瑚缸，是海水缸的进阶目标。

**核心理念：** 追求珊瑚的最高状态，颜色鲜艳、生长旺盛。

### 4.5.2 适合生物

**SPS硬骨珊瑚：**

| 品种 | 特点 | 价格 |
|------|------|------|
| 鹿角珊瑚 | 最受欢迎，品种极多 | 50-500元 |
| 瓦片 | 生长呈片状 | 30-300元 |
| 鸟巢 | 枝条细密 | 50-400元 |
| 茉莉花 | 分枝状 | 40-350元 |
| 分支脑 | 脑状结构 | 80-500元 |
| 叠牙 | 生长快速 | 50-400元 |
| 图钉 | 形态独特 | 60-400元 |
| 瓦片类 | 颜色多样 | 30-200元 |

**适合混养的鱼类：**

- 小丑鱼（少量）
- 海金鱼（少量）
- 极为温和的小型鱼

**不适合混养：**

- 大型鱼（空间不足）
- 会啃珊瑚的鱼
- 领地意识强的鱼

### 4.5.3 环境要求

**光照要求（非常重要）：**

| 参数 | 范围 | 说明 |
|------|------|------|
| PAR | 150-400 | 高光 |
| 光谱 | 全光谱，包含UV | 需含蓝光和UV |
| 光强分布 | 均匀 | 避免热点 |
| 每天时长 | 8-10小时 | 定时开关 |

**水流要求（非常重要）：**

| 参数 | 范围 | 说明 |
|------|------|------|
| 强度 | 强 | 需要高流速 |
| 方向 | 多角度 | 避免死角 |
| 变化 | 有规律变化 | 模拟自然洋流 |
| 设备 | 专业造浪泵 | 多台配合 |

**水质要求（非常严格）：**

| 参数 | 理想范围 | 说明 |
|------|----------|------|
| NO3 | 0.1-5 ppm | 越低越好 |
| PO4 | <0.03 ppm | 越低越好 |
| KH | 8-11 dKH | 维持骨骼生长 |
| Ca | 420-450 ppm | 骨骼必需 |
| Mg | 1300-1400 ppm | 维持钙镁平衡 |
| pH | 8.1-8.4 | 保持稳定 |

### 4.5.4 设备配置

**基础设备：**

| 设备 | 推荐配置 | 预算 |
|------|----------|------|
| 缸体 | 90-150cm | 1000-3000元 |
| 活石 | 0.5-0.8kg/L | 300-1500元 |
| 蛋分 | 处理量3-4倍水体 | 600-1500元 |
| 照明 | 高端全光谱LED | 1500-3000元 |
| 造浪 | 3台以上专业 | 800-2000元 |
| 底缸 | 多格设计 | 800-1500元 |
| 钙反 | 必需 | 500-1000元 |
| 加热/制冷 | 温控系统 | 300-800元 |
| 水质监测 | 试剂或设备 | 200-1000元 |
| **合计** | - | **6000-15000元** |

### 4.5.5 优点与缺点

**优点：**

1. 颜色最鲜艳
2. 状态最佳时非常漂亮
3. 可以观察到明显的生长
4. 在水族箱中展现自然的奇迹

**缺点：**

1. 难度极高
2. 成本高（设备+珊瑚+维护）
3. 对水质要求苛刻
4. 死亡风险大
5. 需要投入大量时间和精力

### 4.5.6 适合人群

- 有丰富海水缸经验的玩家
- 追求极致的玩家
- 有足够时间和精力的玩家
- 预算充足的玩家

**警告：** SPS缸不适合新手，请在积累足够经验后再尝试。

## 4.6 混合型珊瑚缸

### 4.6.1 定义与特点

**定义：** 同时饲养LPS和SPS两种类型的珊瑚。

**核心理念：** 平衡不同珊瑚的需求，创造丰富的视觉效果。

### 4.6.2 挑战与难点

1. **光照平衡**：SPS需要高光，LPS需要中低光，需要找到平衡点
2. **水质平衡**：需要维持在SPS级别，对LPS来说可能过高
3. **珊瑚相容性**：部分LPS会咬SPS，需要合理布局
4. **水流平衡**：SPS需要强水流，LPS需要中低水流

### 4.6.3 建议

1. **先从LPS开始**：积累经验后再尝试混合
2. **分区域管理**：高光区和低光区分开
3. **合理选择品种**：选择对环境要求相近的品种
4. **逐步增加SPS**：从小量开始，逐步增加

## 4.7 其他类型

### 4.7.1 珊瑚礁缸 (Reef Tank)

- **定义**：模拟自然珊瑚礁生态缸，生物多样性高
- **特点**：追求自然生态缸效果
- **设备**：要求高
- **难度**：非常高

### 4.7.2 藻类缸 (ATS)

- **定义**：用大型藻类吸收营养盐的系统
- **作用**：控制NO3和PO4
- **应用**：作为辅助系统配合其他缸

### 4.7.3 生物缸 (BioCube)

- **定义**：一体式小缸，设计紧凑
- **特点**：适合桌面放置
- **适合**：空间有限的玩家

### 4.7.4 海马缸

- **定义**：专门饲养海马的缸
- **特点**：需要特殊设备和活饵
- **难度**：较高

## 4.9 选择建议

### 4.9.1 新手建议

**推荐顺序：**

1. FOT（1-2年经验）
2. FOWLR（2-3年经验）
3. LPS（3年以上经验）
4. SPS（5年以上经验）

### 4.9.2 时间投入

| 类型 | 每周维护时间 |
|------|-------------|
| FOT | 1-2小时 |
| FOWLR | 2-3小时 |
| LPS | 3-5小时 |
| SPS | 5-10小时 |
